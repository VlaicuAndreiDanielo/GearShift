\doxysubsection{Detailed Description}
Scena afișată la sfârșitul jocului (Game Over). 

Desenează un ecran roșu cu un panou „\+GAME OVER” + scorul final. După câteva secunde revine automat în meniul principal sau poate fi schimbată scena la apăsarea Enter / Escape.
