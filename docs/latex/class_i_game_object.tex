\doxysubsection{Detailed Description}
Interfață pentru un obiect de joc utilizat la randare. 

Este un „view simplificat” peste \doxylink{class_game_object}{Game\+Object}, folosit de layer-\/ul de UI pentru a desena obiectele fără să depindă de logica lor internă.
