\doxysubsection{Detailed Description}
Segment de drum pe care rulează mașina. 

Conține un trigger pentru detectarea trecerii Player-\/ului și poate avea atașate gărdulețe de margine ca Game\+Object-\/uri copil. Se repoziționează pentru a simula un drum infinit.
