\doxysubsection{Detailed Description}
Command pentru pauzarea / reluarea jocului. 

Dacă jocul este în starea {\ttfamily Game\+State\+::\+Playing}, comanda îl pune pe pauză. Dacă este în {\ttfamily Game\+State\+::\+Paused}, comanda reia jocul. În alte stări nu are efect.
