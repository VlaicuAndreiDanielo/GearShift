\doxysubsection{Detailed Description}
Manager global pentru coliziuni. 

Deține referințe slabe către toate collider-\/ele înregistrate în scenă. La fiecare apel de \doxylink{class_collision_manager_a4daf19e43a0b0baff43ce02b4b718791}{update()}, curăță collider-\/ele expirate, verifică coliziunile pereche și apelează rezolvarea + notificarea Game\+Object-\/urilor.
