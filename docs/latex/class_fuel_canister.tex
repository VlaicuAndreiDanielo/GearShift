\doxysubsection{Detailed Description}
Obiect de tip „bidon de combustibil” colectabil. 

Când intră în coliziune cu Player-\/ul, dezactivează propriul \doxylink{class_game_object}{Game\+Object} și notifică \doxylink{class_game_logic}{Game\+Logic} pentru a reîncărca fuel-\/ul.
