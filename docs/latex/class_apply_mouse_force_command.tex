\doxysubsection{Detailed Description}
Command pentru aplicarea unei forțe de mouse asupra pânzei (fabric). 

Este folosită pentru interacțiunea cu {\ttfamily \doxylink{class_i_fabric}{IFabric}} (efectul de cloth) din joc, în funcție de poziția mouse-\/ului și dacă butonul este apăsat.
