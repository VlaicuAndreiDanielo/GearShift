\doxysubsection{Detailed Description}
\doxylink{class_renderer}{Renderer} pentru un singur obiect de joc. 

Desenează mai întâi un dreptunghi colorat (hitbox / fundal) și apoi sprite-\/ul asociat, obținut prin {\ttfamily \doxylink{class_sprite_manager}{Sprite\+Manager}} și {\ttfamily \doxylink{class_sprite_mapper}{Sprite\+Mapper}}.
