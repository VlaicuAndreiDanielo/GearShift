\doxysubsection{Detailed Description}
Interfață pentru o scenă din joc (meniu, joc, game over etc.). 

Scenele sunt gestionate de {\ttfamily \doxylink{class_scene_mgr}{Scene\+Mgr}} și reprezintă stări distincte ale aplicației (Main Menu, Game, Game\+Over...).
