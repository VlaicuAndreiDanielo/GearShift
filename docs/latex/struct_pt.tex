\doxysubsection{Detailed Description}
Reprezintă un punct din pânza (cloth) simulată. 

Fiecare punct reține poziția curentă, poziția anterioară (pentru integrare de tip Verlet), forța acumulată și un flag dacă este sau nu „pin-\/uit”.
