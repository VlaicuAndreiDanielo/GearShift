\doxysubsection{Detailed Description}
Command pentru pornirea unei runde noi de joc. 

Verifică dacă jocul se află în starea {\ttfamily Game\+State\+::\+Menu} în {\ttfamily \doxylink{class_start_game_command_a36b2463838de8d011240cd55beaef956}{can\+Execute()}} și, la {\ttfamily \doxylink{class_start_game_command_a78db4c99fbcee401f9472e817f5d05ac}{execute()}}, apelează {\ttfamily \doxylink{class_i_game_af174e89498832afc7458be23a6ebc3a6}{IGame\+::start\+Game()}}. Scene transitions și logica UI sunt tratate separat.
