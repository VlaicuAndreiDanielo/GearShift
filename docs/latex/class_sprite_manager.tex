\doxysubsection{Detailed Description}
Manager simplu de texturi cu caching. 

Încarcă texturi de pe disc la prima cerere și le reține într-\/un cache (map de la path la SDL\+\_\+\+Texture\texorpdfstring{$\ast$}{*}). Oferă și metode helper pentru desen.
