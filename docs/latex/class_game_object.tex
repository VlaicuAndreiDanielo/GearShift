\doxysubsection{Detailed Description}
Bază abstractă pentru toate obiectele din lume. 

Reprezintă un obiect 2D cu poziție, rotație, dimensiune, sprite și tip. Instanțele trebuie gestionate prin std\+::shared\+\_\+ptr.
