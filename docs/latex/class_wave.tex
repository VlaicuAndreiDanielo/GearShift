\doxysubsection{Detailed Description}
Efect de fundal tip „valuri” care folosesc pânza (\doxylink{class_fabric}{Fabric}) ca deformare. 

Construiește o grilă de tile-\/uri colorate, le animă cu funcții trigonometrice în timp și le „agață” de punctele pânzei (\doxylink{class_i_fabric}{IFabric}) pentru a le deforma în funcție de mișcarea cloth-\/ului. \+:content\+Reference\mbox{[}oaicite\+:3\mbox{]}\{index=3\}
