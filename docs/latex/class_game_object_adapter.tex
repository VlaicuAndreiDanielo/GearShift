\doxysubsection{Detailed Description}
Adapter între \doxylink{class_game_object}{Game\+Object} și interfața \doxylink{class_i_game_object}{IGame\+Object}. 

Permite folosirea Game\+Object-\/urilor din logica de joc în layer-\/ul de UI (randare) fără ca acesta să depindă de implementarea concretă a \doxylink{class_game_object}{Game\+Object}.
