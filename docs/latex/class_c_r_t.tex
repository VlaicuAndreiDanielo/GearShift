\doxysubsection{Detailed Description}
Efect vizual tip \doxylink{class_c_r_t}{CRT} (scanlines, radial warp, flicker) peste scenă. 

Randează scena într-\/o textură off-\/screen și apoi aplică\+:
\begin{DoxyItemize}
\item deformare tip „pincushion” (warp spre exterior);
\item scanlines animate;
\item glow / noise / flicker subtil. 
\end{DoxyItemize}
