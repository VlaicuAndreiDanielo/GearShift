\doxysubsection{Detailed Description}
Interfață care descrie starea curentă a input-\/ului de la utilizator. 

Implementările concrete (de ex. din SDL) furnizează informații despre tastele apăsate și despre mouse, astfel încât logica de joc să fie independentă de framework-\/ul de input.
