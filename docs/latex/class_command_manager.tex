\doxysubsection{Detailed Description}
Manager pentru comandă (Command pattern) cu suport de undo. 

Reține un registru de comenzi numite și un stack de undo. Se ocupă de\+:
\begin{DoxyItemize}
\item înregistrarea comenzilor;
\item executarea lor în siguranță (try/catch + logging);
\item gestionarea stack-\/ului de undo până la o dimensiune maximă. 
\end{DoxyItemize}
