\doxysubsection{Detailed Description}
Scena de meniu principal al jocului. 

Afișează fundalul animat (wave + fabric), efectul \doxylink{class_c_r_t}{CRT}, mașina din meniu și butoanele PLAY / EXIT. Folosește Command pattern pentru\+:
\begin{DoxyItemize}
\item start de joc ({\ttfamily \doxylink{class_start_game_command}{Start\+Game\+Command}});
\item ieșire din aplicație ({\ttfamily \doxylink{class_exit_application_command}{Exit\+Application\+Command}}). 
\end{DoxyItemize}
