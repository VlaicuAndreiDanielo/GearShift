\doxysubsection{Detailed Description}
Buton UI simplu folosit în meniuri. 

Desenează un dreptunghi colorat cu text centrat, detectează hover-\/ul și click-\/ul și, la click, execută fie o \doxylink{class_i_command}{ICommand}, fie un callback funcțional.
