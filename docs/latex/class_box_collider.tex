\doxysubsection{Detailed Description}
Colider dreptunghiular (oriented bounding box) pentru un \doxylink{class_game_object}{Game\+Object}. 

Reprezintă un paralelogram 2D atașat unui \doxylink{class_game_object}{Game\+Object}, folosit pentru detecția coliziunilor cu alți Box\+Collider-\/i prin Separating Axis Theorem.
