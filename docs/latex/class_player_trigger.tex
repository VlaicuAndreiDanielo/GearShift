\doxysubsection{Detailed Description}
\doxylink{class_game_object}{Game\+Object} invizibil folosit ca trigger pentru \doxylink{class_player}{Player}. 

Este atașat de obicei la un \doxylink{class_road_segment}{Road\+Segment} sau la un NPC și detectează momentul în care Player-\/ul îl depășește / îl atinge, notificând obiectul părinte.
