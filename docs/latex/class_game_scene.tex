\doxysubsection{Detailed Description}
Scena principală de joc (gameplay). 

Se ocupă de\+:
\begin{DoxyItemize}
\item randarea obiectelor de joc (mașini, drum, fuel etc.);
\item randarea HUD-\/ului (fuel + scor);
\item gestionarea pauzei (overlay „\+PAUSED” + \doxylink{class_pause_game_command}{Pause\+Game\+Command});
\item tranziția către \doxylink{class_game_over_scene}{Game\+Over\+Scene} la final de rundă. 
\end{DoxyItemize}
