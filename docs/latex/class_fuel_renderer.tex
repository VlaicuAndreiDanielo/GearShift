\doxysubsection{Detailed Description}
HUD pentru afișarea fuel-\/ului (timp de joc) + iconiță de canistră. 

Desenează o bară de fundal, o zonă umplută proporțional cu current\+Time / max\+Time și un sprite cu canistra de fuel lângă bară.
