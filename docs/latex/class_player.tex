\doxysubsection{Detailed Description}
Mașina controlată de jucător. 

Gestionează input-\/ul de la utilizator (mișcare, accelerație/frânare), limitele de deplasare pe ecran și notifică listener-\/ii atunci când jucătorul este eliminat.
