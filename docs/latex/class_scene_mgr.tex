\doxysubsection{Detailed Description}
Manager pentru scenele jocului (Menu, Game, Game\+Over etc.). 

Deține un map de scene, știe scena curentă și gestionează tranzițiile într-\/un mod sigur (tranziția se face la începutul lui \doxylink{class_scene_mgr_acbc99da433c225dae4489b2f59367684}{update()}).
