\doxysubsection{Detailed Description}
Thin wrapper peste fereastra și renderer-\/ul SDL. 

Se ocupă de inițializarea SDL (video, TTF, PNG), crearea ferestrei și a renderer-\/ului hardware-\/accelerat, plus operațiile de clear/present.
