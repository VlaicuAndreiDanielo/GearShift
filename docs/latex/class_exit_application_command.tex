\doxysubsection{Detailed Description}
Command pentru ieșirea din aplicație. 

Trimite un eveniment {\ttfamily SDL\+\_\+\+QUIT} în coada de evenimente SDL, lăsând bucla principală a aplicației să facă shut-\/down-\/ul controlat.
