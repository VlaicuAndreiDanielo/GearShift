Detailed Description

Mașină NPC de bază din trafic.

Se deplasează pe drum cu o viteză aleatoare în intervalul [minSpeed, maxSpeed], se repoziționează (respawn) în fața camerei când este depășită de jucător și folosește trigger-e suplimentare pentru a încetini în funcție de trafic. :contentReference[oaicite:1]{index=1}

Creează un NPC de trafic și obiectele asociate (trigger-e).

Creează:

  • NPC-ul propriu-zis;
  • un PlayerTrigger pentru respawn;
  • un SlowDownTrigger pentru încetinire în funcție de trafic;
  • atașează un BoxCollider NPC-ului.
