Detailed Description

Colider dreptunghiular (oriented bounding box) pentru un GameObject.

Reprezintă un paralelogram 2D atașat unui GameObject, folosit pentru detecția coliziunilor cu alți BoxCollider-i prin Separating Axis Theorem.
