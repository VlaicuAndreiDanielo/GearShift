Detailed Description

Implementare de pânză (cloth) 2D bazată pe masă–arc.

Reprezintă o rețea de puncte conectate care se comportă ca un material flexibil. Poate fi actualizată în timp, perturbată cu forțe locale și resetată.

Aplică o forță locală asupra pânzei.

Toate punctele aflate într-un cerc de rază radius în jurul (mx, my) vor primi o forță proporțională cu distanța până la centru.
