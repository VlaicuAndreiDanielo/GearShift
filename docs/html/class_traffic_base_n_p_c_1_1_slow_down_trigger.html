Detailed Description

Trigger intern folosit pentru încetinirea NPC-ului.

Detectează prezența altor mașini (PLAYER sau NPC) în față și, la coliziune, apelează TrafficBaseNPC::stop() pe NPC-ul părinte. :contentReference[oaicite:2]{index=2}
