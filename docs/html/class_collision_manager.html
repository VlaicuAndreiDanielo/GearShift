Detailed Description

Manager global pentru coliziuni.

Deține referințe slabe către toate collider-ele înregistrate în scenă. La fiecare apel de update(), curăță collider-ele expirate, verifică coliziunile pereche și apelează rezolvarea + notificarea GameObject-urilor.
