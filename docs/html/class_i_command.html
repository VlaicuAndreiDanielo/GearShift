Detailed Description

Interfață de bază pentru toate comenzile (Command pattern).

Comenzile pot fi executate, opțional anulate (undo), și furnizează o descriere textuală pentru logging și debugging.

Verifică dacă comanda poate fi executată în starea curentă.

Implementările pot valida contextul (ex. starea jocului). Implicit true.

Indică dacă această comandă suportă operația de undo.

Implicit false. Dacă o comandă suprascrie undo(), ar trebui să suprascrie și canUndo() pentru a întoarce true.
