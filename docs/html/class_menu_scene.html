Detailed Description

Scena de meniu principal al jocului.

Afișează fundalul animat (wave + fabric), efectul CRT, mașina din meniu și butoanele PLAY / EXIT. Folosește Command pattern pentru:

  • start de joc (StartGameCommand);
  • ieșire din aplicație (ExitApplicationCommand).
